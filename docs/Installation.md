# Installation

Recommended: build from source (requires Rust + Cargo):

```bash
git clone https://github.com/ans-inayat/zyraxbuster.git
cd zyraxbuster
cargo build --release
# Binary at target/release/zyraxbuster
```

Or install via cargo:

```bash
cargo install --path .
```

See the main README for full installation options and releases:
https://github.com/ans-inayat/zyraxbuster/blob/main/README.md
